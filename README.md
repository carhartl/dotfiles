# dotfiles

## Requirements

- Git
- Curl

## Install

```bash
curl -Lks https://git.io/JSg3X | /bin/bash
```

## Optional

Add local git configuration:

```bash
cp .gitconfig.local{.example,}
```

To add additonal, local-only configuration:

```bash
cp .localrc{.example,}
```

^ Adapt the placeholders!
